# Agent Nexus — Frontend

Phase 1 mock UI: all 7 pages with full interactions, data from in-memory mock.

## Prerequisites

- Node.js 22+
- pnpm 11+

## Commands

```bash
# Install dependencies
pnpm install

# Development server (http://localhost:3000)
pnpm dev

# Type check
pnpm typecheck

# Production build (outputs to dist/)
pnpm build
```

## Running from repo root

```bash
pnpm -C src-react install
pnpm -C src-react dev
```
